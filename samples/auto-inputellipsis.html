ACInputEllipsis — 输入框超长省略

当输入框内容超出宽度时以省略号显示，聚焦时展开显示完整内容。

1. 基础省略显示


点击输入框可展开查看/编辑完整内容

<input type="text"
  data-component="inputellipsis"
  value="很长很长的内容...">
<!-- 失焦时自动截断，聚焦时显示完整内容 -->

2. 只读省略（不可编辑）

文件路径
描述 
<input type="text"
  data-component="inputellipsis"
  readonly
  value="非常长的只读内容...">

3. 表单中多字段省略

<!-- 批量为多个输入框添加省略效果 -->
<input data-component="inputellipsis" name="name" value="...">
<input data-component="inputellipsis" name="company" value="...">

说明

工作原理：ACInputEllipsis 通过 CSS text-overflow: ellipsis 实现省略显示（失焦状态）， 聚焦时去除省略样式以便查看和编辑完整内容，失焦后恢复省略效果。 适合用于弹窗表格、详情页等空间有限的场景。
