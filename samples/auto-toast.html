ACToast — 声明式 Toast 消息

通过 data-component="toast" 为元素绑定点击弹出 Toast 通知。

1. 基础类型

<button data-component="toast"
  data-toast-message="操作成功！"
  data-toast-type="success">
  点击弹出
</button>

<!-- type 可选值：success | warning | error | info（默认）-->

2. 自定义持续时间

<button data-component="toast"
  data-toast-message="将在 5 秒后消失"
  data-toast-timeout="5000"
  data-toast-type="info">
  5 秒 Toast
</button>

3. 与 JS Toast API 对比

import { Toast } from '../src/Widget/Toast.js';

// JS 主动调用（等同于声明式效果）
Toast.show('成功消息', Toast.TYPE_SUCCESS, 3000);
Toast.show('错误消息', Toast.TYPE_ERROR);

data-* 属性说明

属性可选值说明
data-toast-message任意字符串Toast 显示的文本内容
data-toast-typesuccess / warning / error / info消息类型，影响颜色和图标
data-toast-timeout毫秒数字字符串显示持续时长，默认 3000ms
