ACDialog — 声明式对话框

通过 data-component="dialog" 声明式绑定，点击元素自动弹出对话框。

1. 内嵌内容对话框

<button
  data-component="dialog"
  data-dialog-content="<p>内容文本</p>"
  data-dialog-title="对话框标题">
  点击弹出
</button>

2. 加载外部 URL 内容

加载 HTML 片段（a 标签） 或 
<!-- 使用 a 标签，href 会被 data-dialog-url 覆盖 -->
<a href="#"
   data-component="dialog"
   data-dialog-url="/fragment/detail.html"
   data-dialog-title="详情">查看详情</a>

3. iframe 模式

<button
  data-component="dialog"
  data-dialog-url="/page/edit"
  data-dialog-iframe="1"
  data-dialog-title="编辑"
  data-dialog-width="800"
  data-dialog-height="600">编辑</button>

4. 尺寸与样式控制

<button
  data-component="dialog"
  data-dialog-content="..."
  data-dialog-width="700"
  data-dialog-height="500">打开</button>

data-* 属性说明

属性说明
data-dialog-content直接嵌入的 HTML 内容（与 url 二选一）
data-dialog-url异步加载的 URL（AJAX 加载内容）
data-dialog-title对话框标题
data-dialog-width宽度（像素，数字）
data-dialog-height高度（像素，数字）
data-dialog-iframe为 "1" 时以 iframe 方式加载 url
